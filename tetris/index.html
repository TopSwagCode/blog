Tetris Wallpaper Generator
A small project I made because I wanted to change wallpaper on my PC. So I created this tool to create a Tetris themed wallpaper for any screen size. Needed it to be flexible I use a range of different screen resolutions. Eg. 4k, Ultrawidescreen and HD. So I wanted to create a different unique background for each screen.
